🔗 Підключення Apps Script
Вставте сюди URL задеплоєного веб-застосунку Apps Script.
Це значення зберігається локально у вашому браузері для зручності.
📋 Інструкція налаштування
  1. Відкрийте Google Sheets → Розширення → Apps Script
  2. Замінить вміст на скопійований код нижче → збережіть
  3. Розгортання → Нове розгортання → Веб-застосунок
  4. Виконувати як: Я · Доступ: Усі (навіть анонімні)
  5. Натисніть Розгорнути → скопіюйте URL
  6. Вставте URL вище, а також у константу SCRIPT_URL у form.html та admin.html
🚀 GitHub Pages
  1. Створіть репозиторій на GitHub
  2. Завантажте form.html та admin.html в корінь репо
  3. Settings → Pages → Branch: main → Save
  4. Посилання на форму: https://username.github.io/repo/form.html
  5. Посилання на адмінку: https://username.github.io/repo/admin.html
